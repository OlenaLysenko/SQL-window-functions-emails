# SQL Window Functions – Emails Analysis

## Опис проєкту
У цьому проєкті використано window functions для аналізу відправки імейлів.

Розраховується:
- відсоток імейлів, відправлених кожному акаунту за місяць від загальної кількості
- дата першого надсилання імейлу в межах місяця
- дата останнього надсилання імейлу в межах місяця

---

## Використані функції
- COUNT() OVER()
- `MIN() OVER()`
- `MAX() OVER()`
- `PARTITION BY`
